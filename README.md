# 📁 File Type Analyzer (Windows Forms App)

## 📌 Description

This application allows the user to select a folder and analyze its contents.
It counts:

* Total number of files
* Total number of folders
* Number of files by type (e.g., .txt, .jpg, .pdf)

The program also scans all subfolders.

---

## ⚙️ Features

* Browse and select a folder
* Count files and folders
* Traverse subdirectories
* Display file types with counts

---

## 🖥️ Technologies Used

* C#
* Windows Forms (.NET)
